Common Properties and Methods are not Listed Here.

Properties :

1)    Mode :
        Setting Startup Mode.
    a) PickerHue
    b) PickerSaturation
    c) PickerBrightness
    d) CustomColors
    e) SafeColor.
2)    DialogStartUpPosition :
        Sets where the Dialog would appear.
    a) CentreOwner
    b) CentreScreen
    c) Manual.

3)    Color :
        Sets Initial Color or Get Selected Color.
4)    Icon :
       User can set the Icon of Dialog.
       Or it can left blank.
5)    DialogLeft and DialogTop :
        Set the Position of Dialog when
       'DialogStartUpPosition' is Set to 'Manual'
6)    AdjustPosition :
        If set True will adjust Dialog position
       such that it will remain within the Screen
7)    WebColors
       Sets or returns that the dialog shows only web safe colors.

Methods :

1)    Show :
        Sets the Dialog visible.
2)    Refresh:
        This Unloads the current Dialog and
        reloads it fresh.

Events:

1)    ColorChanged. (With NewColor parameter)
       Raises this event dynamically when user changes the selected color
